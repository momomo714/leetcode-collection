题目列表

1\. 括号平衡检验

文件：括号平衡.cpp、括号平衡.md

题目：判断一段代码中的括号 ()\[]{}是否正确匹配且嵌套正确。

解法：利用栈（stack）的“后进先出”特性，遍历字符串，左括号入栈，右括号与栈顶匹配，最后检查栈是否为空。

运行：编译后输入代码（以 EOF 结束），程序输出“正确”或“错误”。

&#x20;2. 最长无重复子串

文件：最长无重复字符串.cpp、最长无重复字符串.md

题目：给定一个字符串 s，找出其中不含有重复字符的最长子串的长度。

解法：滑动窗口 + 哈希表。用两个指针 left和 right 维护窗口，哈希表记录字符最后出现的位置，遇到重复时移动左指针，窗口长度即为当前无重复子串长度，取最大值。

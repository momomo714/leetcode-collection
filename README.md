题目列表

1\. 括号平衡检验

文件：括号平衡.cpp、括号平衡.md

题目：判断一段代码中的括号 ()\[]{}是否正确匹配且嵌套正确。

解法：利用栈（stack）的“后进先出”特性，遍历字符串，左括号入栈，右括号与栈顶匹配，最后检查栈是否为空。

运行：编译后输入代码（以 EOF 结束），程序输出“正确”或“错误”。

&#x20;2. 最长无重复子串

文件：最长无重复字符串.cpp、最长无重复字符串.md

题目：给定一个字符串 s，找出其中不含有重复字符的最长子串的长度。

解法：滑动窗口 + 哈希表。用两个指针 left和 right 维护窗口，哈希表记录字符最后出现的位置，遇到重复时移动左指针，窗口长度即为当前无重复子串长度，取最大值。

3\. 两个正序数组中位数

文件：两个正序数组中位数/两个正序数组中位数.cpp题目：给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的中位数。

解法：二分法。在较短的数组上二分分割线位置，使左侧元素总数等于 (m+n+1)/2，并满足左侧最大值 ≤ 右侧最小值，然后根据总长度奇偶性返回中位数。

运行：编译后输入两个数组（具体输入格式见代码注释），程序输出中位数。

4\. Z 字形变换

文件：Z字形变换字符串.cpp、Z字形变换字符串.md

题目：将一个给定字符串 s 根据给定的行数 numRows，以从上往下、从左到右进行 Z 字形排列，然后按行输出变换后的字符串。

解法：按行访问。先计算周期长度 cycle = 2 \* numRows - 2。对于每一行 i，首先添加行首字符 s\[i]，然后按周期添加后续字符；对于非首尾行，还需在每周期中添加斜线上的字符（索引 j + cycle - 2 \* i）。注意边界判断和 numRows == 1 的特殊情况。

5.5. 字符串转换整数 (atoi)

文件：字符串转换整数.cpp、字符串转换整数.md

题目：实现一个 myAtoi 函数，将字符串转换为 32 位有符号整数。函数会丢弃前导空格，然后处理可选的 '+' 或 '-' 符号，接着读取连续的数字字符，将其转换为整数。如果数字部分超出 32 位有符号整数范围 `\[−2^31, 2^31−1]`，则截断到边界值；如果没有数字则返回 0。

解法：按步骤模拟。用索引遍历字符串：① 跳过空格；② 解析符号；③ 累加数字（用 long long 暂存，防止溢出）；④ 每次累加后检查是否超出边界；⑤ 返回带符号的整数。注意处理空字符串、只有符号、溢出等边界情况。

运行：编译后输入字符串，程序输出转换后的整数（示例输入："   -42" → 输出 -42）
